Temat 5: Wariancje

Wariancja liczb \(x_1, x_2, \ldots, x_n\) to średnia arytmetyczna kwadratów odchyleń od ich średniej arytmetycznej: \[ \sigma^2 = \frac{(x_1 - \overline{X})^2 + (x_2 - \overline{X})^2 + \ldots + (x_n - \overline{X})^2}{n} \]

Przykład 1. Obliczymy wariancję liczb \(x_1 = 4, x_2 = -3, x_3 = 2\). Najpierw policzymy średnią arytmetyczną: \[ \overline{X} = \frac{4 + (-3) + 2}{3} = 1 \] Zatem: \[ \sigma^2 = \frac{(4-1)^2 + (-3-1)^2 + (2-1)^2}{3} = \frac{9 + 16 + 1}{3} = \frac{26}{3} \]

Zastosowania wariancji:

  • Wariancja jest używana do pomiaru zmienności danych. Jest to szczególnie ważne w statystyce, gdzie pomaga w analizie rozkładu danych.
  • W finansach wariancja służy do oceny ryzyka związanego z inwestycjami. Większa wariancja oznacza większe ryzyko.
  • Wariancja jest podstawą do dalszych analiz statystycznych, takich jak obliczanie odchylenia standardowego i testów statystycznych.
  • Wariancja jest wykorzystywana w kontrolowaniu jakości, gdzie pomaga monitorować stabilność procesów produkcyjnych.

Zadanie 1. Każdy z trzech zawodników wykonał serię 13 rzutów do kosza. Liczba rzutów celnych wynosiła odpowiednio 5,7 oraz 9. Oblicz wariancję celnych rzutów do kosza.

Liczba rzutów w serii (\(13\)) nas nie interesuje. Aby obliczyć wariancję, musimy zacząć od znalezienia średniej arytmetycznej: \[ \overline{X} = \frac{5 + 7 + 9}{3} = 7 \] Zatem wariancja wynosi: \[ \sigma^2 = \frac{(5-7)^2 + (7-7)^2 + (9-7)^2}{3} = \frac{4 + 0 + 4}{3} = \frac{8}{3} \]

Zadanie 2. Oblicz wariancję zestawu liczb \(5, 2, 9, 8, 5, 9, 11, 8\)

Najpierw obliczamy średnią arytmetyczną: \[ \overline{X} = \frac{5 + 2 + 9 + 8 + 5 + 9 + 11 + 8}{8} = \frac{57}{8} = 7.125 \] Następnie obliczamy wariancję: \[ \sigma^2 = \frac{(5-7.125)^2 + (2-7.125)^2 + (9-7.125)^2 + (8-7.125)^2 + (5-7.125)^2 + (9-7.125)^2 + (11-7.125)^2 + (8-7.125)^2}{8} \] \[ = \frac{4.515625 + 26.265625 + 3.515625 + 0.765625 + 4.515625 + 3.515625 + 15.015625 + 0.765625}{8} \] \[ = \frac{58.875}{8} = 7.359375 \]
